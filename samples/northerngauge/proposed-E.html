Thread Standards

API 5B vs API 7-2: When Each Standard Applies

API 5B covers casing, tubing, and line pipe threads. API 7-2 addresses rotary shouldered connections. Knowing which applies determines your gauge selection, inspection protocol, and traceability chain.

NIST Traceability: What the Chain Actually Means

Traceability is not a certification phrase. It is a documented, unbroken chain linking your gauge to a national measurement standard. Understanding it protects against audit failures and measurement disputes.
